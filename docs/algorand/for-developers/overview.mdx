# Milkomeda A1 Overview


## Public Endpoints

Milkomeda A1 offers **HTTPS** and **WebSocket** endpoints for both Mainnet and Devnet, which are publicly accessible to users and developers.



### Mainnet
> **Network Name:** Milkomeda Algorand (A1)
>
> **HTTPS RPC URL:** https://rpc-mainnet-algorand-rollup.a1.milkomeda.com
>
> **WebSocket RPC URL:** wss://rpc-mainnet-algorand-rollup.a1.milkomeda.com/ws
>
> **Chain ID:** 2002
>
> **Currency Symbol:** mALGO
>
> **Block Explorer URL:** https://explorer-mainnet-algorand-rollup.a1.milkomeda.com
> <button onClick={() => changeNetwork('a1')}  className='button button--secondary'>Add A1 Mainnet to Metamask</button>



### Devnet
> **Network Name:** Milkomeda Algorand Devnet
>
> **New RPC URL:** https://rpc-devnet-algorand-rollup.a1.milkomeda.com
>
> **WebSocket RPC URL:** wss://rpc-devnet-algorand-rollup.a1.milkomeda.com/ws
>
> **Chain ID:** 200202
>
> **Currency Symbol:** mTALGO
>
> **Block Explorer URL:** https://testnet-algorand-rollup.a1.milkomeda.com
> <button onClick={() => changeNetwork('a1_testnet')}  className='button button--secondary'>Add A1 Devnet to Metamask</button>




## Bridge & Explorers

- [Blockscout Explorer](https://explorer-mainnet-algorand-rollup.a1.milkomeda.com/) - Blockscout is a tool for inspecting and analyzing EVM based blockchains
- [Milkomeda Bridge](https://algorand-bridge.milkomeda.com/mainnet) - a UI to bridge between Algorand and Milkomeda A1
- [Milkomeda Bridge Explorer](https://bridge-explorer.milkomeda.com/algorand-mainnet) - a tool to analyze bridge transactions


## Wallets that support A1

- [Metamask](https://metamask.io/) - MetaMask is a popular self-custodial wallet. A safe and simple way to access blockchain applications and web3
