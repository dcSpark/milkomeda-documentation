# Milkomeda C1 Overview

## Bridge & Explorers

- [Blockscout Explorer](https://explorer-mainnet-cardano-evm.c1.milkomeda.com/) - Blockscout is a tool for inspecting and analyzing EVM based blockchains
- [Milkomeda Bridge](https://cardano-bridge.milkomeda.com/) - a UI to bridge from Milkomeda C1 (sidechain) to Cardano (mainchain)
- [Milkomeda Bridge Explorer](https://bridge-explorer.milkomeda.com/cardano-mainnet) - a tool to analyze bridge transactions

## Wallets that support C1

- [Flint Wallet](https://flint-wallet.com/) - Flint's lightweight and user-friendly design lets you manage your crypto assets and access the world of DeFi with ease.
- [NuFi](https://nu.fi/) - Non-custodial wallet for several chains, including Cardano and Milkomeda C1 (+HW wallet support)
- [Metamask](https://metamask.io/) - MetaMask is a popular self-custodial wallet. A safe and simple way to access blockchain applications and web3
- [Gnosis Safe](https://safe.milkomeda.com/) - Milkomeda has deployed an instance of Gnosis Safe to both the Milkomeda C1 and A1 networks. This deployment of Safe brings secure and battle-tested, multisig wallet smart contracts to the Milkomeda networks. Check out this [guide](https://medium.com/@milkomedafoundation/creating-a-gnosis-safe-on-milkomeda-guide-9cd00b6ea62d) for more information.
