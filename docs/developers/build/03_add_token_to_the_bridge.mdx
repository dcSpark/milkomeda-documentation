# Add token to the Bridge


The Milkomeda Bridge Token Listing Request Form is for token issuers to add their token to the Milkomeda bridge. The form requests info about the token, project, and tech specs. 

[Milkomeda Bridge Token Listing Request Form](https://airtable.com/shr8k4usXQFhtDjJf)


The bridge team will review the info and assess suitability for integration.

:::info
Please note that submitting an application does not guarantee the token will be added. 
:::
