解开包装资产

要将资产从Milkomeda C1侧链转移回Cardano，请使用Milkomeda Cardano Unwrap Bridge。 按照以下步骤，解开milkADA的包装，在你的Cardano mainnet钱包中收到ADA。

首先，访问 Milkomeda Cardano Unwrap Bridge 并连接您的 MetaMask 钱包。

点击 「下一步」

连接到 Milkomeda C1 侧链

:::警告 记住在你的钱包中选择适当的网络。
只连接到您信任的网站。 请确保网站URL如下：https://cardano-bridge.milkomeda.com :::

接下来，输入要解包的milkADA的数量，或点击箭头选择不同的资产解包，并在「收件人」栏贴上你的Cardano钱包地址。 然后，点击「下一步」按钮。

点击 「确认」来签名交易，解包MetaMask的资产。

接着，等待区块链确认。

info

你可以在Cardanoscan explorer中检查交易细节，以了解更多信息。

在这个阶段，你的浏览器的地址栏将包含一些关于解包交易的信息。 下图显示了一个例子。

地址栏中的这些元素的含义如下：

  • EVM 解包交易 ID
  • 未包装的代币金额
  • 被解开的代币的名称

要查询有关解包交易的更多详细信息，请进入Milkomeda Blockscout explorer，将EVM解包交易ID贴上 到搜索框中，如下图所示。

查询示例交易

现在解包过程已经完成。 在等待必要的交易确认后，你应该在你的Cardano Mainnet钱包中看到解开的资产。

:::info 要检查可用的可解包的代币列表，请到Cardano Bridge Supported Native Assets部分。 :::
