Obtaining Test ALGO

info

Milkomeda A1 Rollup is fully operational on Mainnet, which means that it is currently deployed and connected to production version of the Algorand blockchain.

If you are familiar with the EVM stack, you can start interacting with the Milkomeda A1 Rollup on Mainnet right away. However, we recommend starting with the devnet to check your deployments. To start we recommend getting some testnet ALGO from the official testnet dispenser https://testnet.algoexplorer.io/dispenser. Also currently we recommend using Pera Wallet https://web.perawallet.app to setup a testnet wallet compatible with the Milkomeda A1 bridge (required to move funds between Algorand testnet and the Rollup).

After waiting a few minutes (usually 2 minutes or less), you should be able to see some tALGO in your Algorand Testnet wallet.

Next we will set up MetaMask and get an address on the Milkomeda A1 Algorand Rollup Mainnet.
