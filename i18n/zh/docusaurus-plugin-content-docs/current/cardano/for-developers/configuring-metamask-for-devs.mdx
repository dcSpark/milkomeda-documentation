# 设置MetaMask

现在我们有一个Cardano测试网钱包和一些测试用ADA，我们需要一个 Milkomeda Cardano Testnet侧链的地址，在那里我们可以收到我们的MilkTADA。 目前，我们建议使用MetaMask来完成这一步骤。

首先，安装[Chrome浏览器的MetaMask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn)。 然后，点击Chrome扩展程序图标 <img src={puzzleIcon} className="img-icon" /> 并选择MetaMask。 在MetaMask窗口，点击「**开始**」然后,「**创建一个钱包**」。 设置钱包密码，并点击完成其余的设置屏幕。 完成后，你应该如下图所示，有一个新的空钱包。 选择MetaMask屏幕右上方的网络下拉菜单。

![](/img/metamask-change-network.png)

从下拉列表中，选择「**添加网络**」，并输入以下内容：

> Network Name: Milkomeda Cardano Testnet
> 
> <p>New RPC URL: https://rpc-devnet-cardano-evm.c1.milkomeda.com</p>
> 
> Chain ID: 200101
> 
> Currency Symbol (Optional): milkTADA
> 
> <p> Block Explorer URL (Optional): https://explorer-devnet-cardano-evm.c1.milkomeda.com </p>
如果你需要一个WebSocket端点，请使用：wss://rpc-devnet-cardano-evm.c1.milkomeda.com

:::info

如果要使用Milkomeda C1的主网版本，请输入以下参数代替：

- Network Name: Milkomeda Cardano (C1)
- <span>New RPC URL: https://rpc-mainnet-cardano-evm.c1.milkomeda.com</span>
- Chain ID: 2001
- Currency Symbol: milkADA
- <span> Block Explorer URL: https://explorer-mainnet-cardano-evm.c1.milkomeda.com </span>

如果你需要一个WebSocket端点，请使用以下内容：wss://rpc-mainnet-cardano-evm.c1.milkomeda.com :::

点击「**保存**」，你现在应该在网络下拉菜单中看到「**Milkomeda Cardano Testnet**」。 点击账户名称，将您的Milkomeda Cardano Testnet钱包地址复制到剪贴板上，如图所示。

![Metamask 复制地址](/img/metamask-copy-c1-testnet-address.png)

接下来我们将回到Flint钱包，将我们的测试ADA送去包装。
