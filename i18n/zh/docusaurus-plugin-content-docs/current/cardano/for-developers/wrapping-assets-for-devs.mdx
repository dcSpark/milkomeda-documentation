# 包装资产

打开Flint钱包窗口，点击「**发送**」。 在下一个屏幕，启用「**Milkomeda模式**」切换，将您的Milkomeda C1钱包地址贴上到「**Milkomeda以太坊地址**」区域，并输入要发送的金额(本例中为300 Test ADA)。 点击**继续**。

![](/img/flint-wrap-tada.png)

在确认屏幕上，输入你的钱包密码并点击「**发送**」。

![](/img/flint-wrap-tada-confirm.png)

转移将需要10次确认，这应该需要大约**5**分钟。 一旦交易被确认，在您的MetaMask的Milkomeda Cardano Testnet钱包中应该可见到转移的MilkTADA。

:::info 关于交易费用的备注

对于仅在Milkomeda C1侧链上进行的交易，您只需支付由区块生产商设定的标准Milkomeda网络交易费用。

然而，涉及包装的交易，即在Cardano和Milkomeda C1侧链之间移动资产，需要为Cardano和Milkomeda支付交易费用，因为每条链上都需要进行一次交易。

例如，从Cardano向Milkomeda C1发送资产需要在Cardano上进行一笔交易来锁定你的资产，在Milkomeda C1上进行另一笔交易来向你发放包装好的资产。

当在Flint钱包中发送这种类型的交易时，你可以展开「网络费用」项目，看到一个 「基本费用」（如Cardano交易费用）和一个 「Milkomeda费用」。 请注意，这些交易的「Milkomeda费」将略高于Milkomeda的标准交易费，因为您也要向区块生产商支付他们在包裹您的资产方面的作用。

:::
