# 配置Flint钱包

Flint Wallet是一个Chrome扩展程序，作为DeFi和NFT的常用轻型钱包。 当前版本的Flint允许用户创建一个Cardano Testnet钱包，并启用 "Milkomeda模式"，将交易发送到Milkomeda C1侧链。

在使用Chrome浏览器时访问以下链接，并点击 「添加到Chrome 」按钮，即可安装Flint Wallet: [安装Flint 钱包](https://chrome.google.com/webstore/detail/flint/hnhobjmcibchnmglfbldbfabcgaknlkj)​

安装后，点击Chrome窗口右上方的扩展程序图标<img src={puzzleIcon} className="img-icon" />，选择**Flint钱包**。 在Flint窗口中，通过最初的警告屏幕，点击 「**让我开始**」，选择你的语言，点击「**我同意**」，当你到达下图所示屏幕时，点击「**Testnet**」按钮。

:::info
要在Milkomeda C1的主网版本上使用真正的ADA，请点击 "主网"，创建一个主网的Cardano钱包来代替。
:::

![选择测试网网络](/img/flint-select-testnet.png)

接下来，点击「**确认**」，「**创建一个新钱包**」，并写下你的恢复短语的副本。

在下一个屏幕上确认恢复短语，设置一个钱包密码，并点击「**创建我的钱包**」。 现在你可以在应用程序密码界面选择「**跳过**」，然后点击「**完成**」。 你现在应该有一个Cardano Testnet钱包，如图所示：

![Flint home screen](/img/flint-home-screen.png)

接下来，我们将访问Cardano Testnet龙头，获得一些测试ADA。
