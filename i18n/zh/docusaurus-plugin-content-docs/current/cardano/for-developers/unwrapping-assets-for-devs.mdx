# 解开包装资产

Milkomeda Cardano Unwrap Bridge是一个将资产从Milkomeda Cardano Testnet转移到Cardano主网的工具。 按照以下步骤，解开milkTADA的包装，在你的Cardano Testnet钱包中收到tADA。

首先，访问 [Milkomeda Cardano Unwrap Bridge for Devnet](https://devnet-cardano-bridge.milkomeda.com/bridge) 并连接您的 MetaMask 钱包。

![](/img/bridge-connect-metamask.png)

![](/img/bridge-connect-metamask-tada-confirm-1.png)

![](/img/bridge-connect-metamask-tada-confirm-2.png)

接下来，输入要解开的milkADA的数量，并在 「收件人」栏贴上你的Cardano Testnet钱包地址。 然后，点击「下一步」按钮。

![](/img/bridge-enter-tada-amount.png)

在 MetaMask 中确认操作。

![](/img/bridge-unwrap-tada-confirm.png)

在等待必要的交易确认约3至5分钟后，你应该在你的Cardano Testnet钱包中看到未包装的tADA。
