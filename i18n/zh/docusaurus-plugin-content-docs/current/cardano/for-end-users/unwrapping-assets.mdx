# 解开包装资产

要将资产从Milkomeda C1侧链转移回Cardano，请使用Milkomeda Cardano Unwrap Bridge。 按照以下步骤，解开milkADA的包装，在你的Cardano mainnet钱包中收到ADA。 &#x20;

首先，访问 [Milkomeda Cardano Unwrap Bridge](https://cardano-bridge.milkomeda.com/bridge) 并连接您的 MetaMask 钱包。

![连接MetaMask](/img/connect-metamask.png)

![连接钱包](/img/connect-wallet.png)

_点击_ 「下一步」

![连接 Milkomeda 网络](/img/connect-milkomeda.png)

_连接到 Milkomeda C1 侧链_

:::警告 记住在你的钱包中选择适当的网络。 <br /> 只连接到您信任的网站。 请确保网站URL如下：https://cardano-bridge.milkomeda.com :::

接下来，输入要解包的milkADA的数量，或点击箭头选择不同的资产解包，并在「收件人」栏贴上你的Cardano钱包地址。 然后，点击「下一步」按钮。

<img className="img-full" src={tokenSelection} />

点击 「确认」来签名交易，解包MetaMask的资产。

![设置MetaMask](/img/confirm-metamask.png)

接着，等待区块链确认。

<img className="img-full" src={blockchainConfirmation} alt="区块链确认" />

:::info

你可以在[Cardanoscan](https://cardanoscan.io) explorer中检查交易细节，以了解更多信息。

:::

在这个阶段，你的浏览器的地址栏将包含一些关于解包交易的信息。 下图显示了一个例子。

<img className="img-full" src={addressBar} alt="地址栏链接" />

地址栏中的这些元素的含义如下：

- EVM 解包交易 ID
- 未包装的代币金额
- 被解开的代币的名称

要查询有关解包交易的更多详细信息，请进入[Milkomeda Blockscout explorer](https://explorer-mainnet-cardano-evm.c1.milkomeda.com/)，将EVM解包交易ID贴上 到搜索框中，如下图所示。

![Flint 交易页面](/img/flint-transactions.png)
