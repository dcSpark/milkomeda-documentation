# Cardano Bridge 支持的本地资产

本节介绍Milkomeda Bridge接Cardano支持的所有本地代币。 要包装和解除任何受支持的本地资产，请按照[給最终用户](./)部分介绍的_步骤_教程，将`milkADA`替换为下表中的任何标记。

:::info
请记住，特定的本地资产必须被添加到Milkomeda Token Registry，才能用于任何包装或解包操作。 因此，该列表是有限的，只允许特定的本地代币。
:::
