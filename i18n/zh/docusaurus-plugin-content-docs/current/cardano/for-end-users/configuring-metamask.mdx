# 设置MetaMask

下一步是在Milkomeda C1侧链上获得一个地址，我们可以在那里接收我们的MilkADA。 目前，我们建议使用MetaMask来完成这一步骤。

首先，安装[Chrome浏览器的MetaMask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn)。 然后，点击Chrome扩展程序图标 <img src={puzzleIcon} className="img-icon" /> 并选择MetaMask。 在MetaMask窗口，点击「**开始**」然后,「**创建一个钱包**」。 设置钱包密码，并点击完成其余的设置屏幕。 完成后，你应该有一个新的空钱包，如下图所示。 选择MetaMask屏幕右上方的网络下拉菜单。

![MetaMask 钱包](/img/metamask-wallet.png)

从下拉列表中，选择「**添加网络**」，并输入以下内容：

> Network Name: Milkomeda Cardano (C1)
> 
> <p> New RPC URL: https://rpc-mainnet-cardano-evm.c1.milkomeda.com </p>
> 
> Chain ID: 2001
> 
> Currency Symbol (Optional): MilkADA
> 
> <p>Block Explorer URL (Optional): https://explorer-mainnet-cardano-evm.c1.milkomeda.com </p>
""， "Milkomeda Cardano（C1）"。 点击「**保存**」， 你现在应该在网络下拉菜单中看到「**Milkomeda Cardano (C1)**」。 如图所示，通过点击账户名称，将你的C1钱包地址复制到剪贴板。

![在MetaMask 中复制地址](/img/copy-address-metamask.png)

现在我们准备返回Flint钱包，将我们的ADA转换为MilkADA。 &#x20;
