# 开始使用Milkomeda

<img src={milkomedaBanner} className="img-full" alt="" />

**Milkomeda** 是一个开创性的新协议，它为非 EVM 区块链提供了EVM 功能。 随着开发进程的推进，Milkomeda将扩展为几个主要区块链：包括卡Cardano、Solana和Algorand提供L2解决方案(rollups)。 目前我们有一个以EVM为基础的侧链 ，已连接到Cardano并运行。 我们把它称为 **Milkomeda C1** 侧链。

按照本指南中的步骤，快速开始使用 Milkomeda。

## 为什么要使用Milkomeda？

- Milkomeda 为非 EVM 生态系统提供了继承关键元素的能力，例如来自基于EVM生态系统的rollup和 Solidity支援。

- 开发者可以使用来自EVM生态系统的所有工具。 建筑、部署、调试和审计所有工作都与以太坊相同。 开发者将项目从以太坊移植过来的能力，将大大拓宽支持的L1区块链上可用的dApps选择。

- 部署之间的安全一致性。多链部署的代码具有相同的安全属性，无需对每个链进行单独审计。

- 未来的零知识技术的创新，将作为L3+解决方案在Milkomeda之上实现。

- Milkomeda使初创企业能够在Cardano、Algorand和Solana等链上建立。 同时利用最大的智能合约社区(Solidity) 的人才和资源。

## Milkomeda的特点

- 可扩展性：Milkomeda可以在不牺牲长期分散性的情况下，扩展到每秒数百个交易。

- 安全性：我们的目标是利用所有区块链的卷积力量，在路线图的关键点进行审计，以确保协议的安全性。

- 用户体验：我们与钱包合作，以确保将资产转移到Milkomeda是轻而易举的。

- 互操作性：Solidity（EVM）的支援将是第一位的，其次是Cairo和其他。

- 可移植性：你只需要编码一次，然后它就可以在不牺牲安全的情况下到处利用。

- 开发者经验：开发者可以直接利用以太坊的所有工具和资源。

## [使用入门：终端用户](./cardano/for-end-users/)

终端用户可以从Cardano主网发送ADA到一个桥接地址，并在 **Milkomeda C1** 侧链上接收包装好的ADA。 跳转到[终端用户入门页面](./cardano/for-end-users)，开始设置Flint和MetaMask。

## [使用入门：：开发者](./cardano/for-developers/)

In addition to the mainnet version of the Milkomeda C1 sidechain which uses wrapped ADA for fees and gas, developers can also access the **Milkomeda Devnet** to wrap assets from the Cardano Testnet and deploy smart contracts on the Milkomeda C1 Devnet sidechain using wrapped Test ADA for fees and gas. Proceed to the next page to start by setting up Flint Wallet and obtaining some test ADA from the Cardano Testnet faucet.
