# 包装资产

进入Milkomeda A1桥页[https://algorand-bridge-dev.milkomeda.com/](https://algorand-bridge-dev.milkomeda.com/)，选择您想移动的代币(在开始时只有tALGO)。 然后连接你的MyAlgo钱包，再连接你的Metamask 最后输入你想转移到Rollup的代币数量。 点击**继续**。

<img src={wrappingAsset} className="img-full" alt="包装资产" />

在确认屏幕上，点击`签名和发送`输入你的钱包密码并确认。 这个进程需要大约20秒钟才能确认。

<img src={wrappingConfirmation} className="img-full" alt="包装确认" />

> 记住，来自Rollup的交易被写入Algorand Testnet区块链。 第一个确认是将资金从钱包发送至桥接地址。第二个确认是对Algorand的Rollup交易的确认。

转移只需要总共2次确认--这应该需要大约20秒(Algorand为4.5秒，Rollup的批次为14秒左右)。 一旦交易被确认，在您的MetaMask的Milkomeda Algorand Testnet钱包中应该可见到转移的MilkTALGO。

<img src={blockchainConfirmation} className="img-full" alt="区块链确认" />

现在，我们已经成功地将测试ALGO包装成MilkTALGO，用于Milkomeda A1 Algorand Rollup Testnet，下一步是部署一个样本智能合约。
