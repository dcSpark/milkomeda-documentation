# 获取测试ALGO

首先，我们建议从官方的测试网分配器 [https://testnet.algoexplorer.io/dispenser](https://testnet.algoexplorer.io/dispenser) 获得一些测试网ALGO。 此外，目前我们建议使用MyAlgo钱包[https://wallet.myalgo.com/](https://wallet.myalgo.com/)来设置一个与Milkomeda A1桥兼容的测试网钱包(需要在Algorand测试网和Rollup之间移动资金)。

<img src={algoDispenser} className="img-full" alt="" />

等待几分钟后(通常2分钟或更短)，你应该能在你的Algorand Testnet钱包中看到一些tALGO。

<img src={algoWalletDashboard} className="img-full" alt="" />
