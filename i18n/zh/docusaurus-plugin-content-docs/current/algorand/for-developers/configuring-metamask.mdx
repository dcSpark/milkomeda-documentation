# 设置MetaMask

现在我们有一个Algorand测试网钱包和一些测试用ALGO，我们需要一个 Milkomeda Algorand Testnet Rollup的地址，在那里我们可以收到我们的MilkTALGO。 目前，我们建议使用MetaMask来完成这一步骤。

首先，安装[Chrome浏览器的MetaMask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn)。 然后，点击Chrome扩展程序图标 <img src={puzzleIcon} alt="" className="img-icon" /> 并选择MetaMask。 在MetaMask窗口，点击「**开始**」然后,「**创建一个钱包**」。 设置钱包密码，并点击完成其余的设置屏幕。 完成后，你应该如下图所示，有一个新的空钱包。 选择MetaMask屏幕右上方的网络下拉菜单。

![](/img/algorand-eth.png)

从下拉列表中，选择「**添加网络**」，并输入以下内容：

> 网络名称: Milkomeda Algorand Testnet
> 
> <p> New RPC URL: https://rpc-devnet-algorand-rollup.a1.milkomeda.com </p>
> 
> Chain ID: 200202
> 
> Currency Symbol (Optional): milkTALGO
> 
> <p> Block Explorer URL (Optional): https://testnet-algorand-rollup.a1.milkomeda.com </p>
点击「**保存**」，你现在应该在网络下拉菜单中看到「**Milkomeda Algorand Testnet**」。 点击账户名称，将您的Milkomeda Algorand Testnet钱包地址复制到剪贴板上，如图所示。

![](/img/algorand-milk.png)

接下来我们将回到Milkomeda A1 bridge，将我们的测试ALGO送去包装。
