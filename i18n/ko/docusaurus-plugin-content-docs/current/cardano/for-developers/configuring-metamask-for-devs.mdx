# 메타마스크 연동

Cardano Testnet 지갑을 만들고 테스트 ADA를 받았으면 Milkomeda Cardano Testnet 사이드 체인에서 MilkTADA를 받기 위한 주소가 필요합니다. 현 단계에서는 MetaMask를 사용하는 것이 좋습니다.

먼저 [Chrome용 MetaMask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn)를 설치합니다. 그 다음에 Chrome 확장 프로그램 아이콘 <img src={puzzleIcon} className="img-icon" />을 클릭하고 MetaMask를 선택합니다. MetaMask 창에서 **Get Started**(시작)를 클릭한 후 **Create a Wallet</ strong>(월렛 만들기)를 선택합니다. 월렛 비밀번호를 설정합니다. 이 후에 설정 화면이 표시되므로 클릭하여 계속 진행합니다. 완료되면 아래와 같은 빈 월렛이 새로 생성됩니다. MetaMask 화면 오른쪽 상단의 네트워크 드롭다운을 선택합니다.</p>

![](/img/metamask-change-network.png)

드롭다운 목록에서 **Add Network**(네트워크 추가)를 선택하고 다음 내용을 입력합니다.

> 네트워크 이름: Milkomeda Cardano 테스트넷
> 
> <p>새 RPC URL: https://rpc-devnet-cardano-evm.c1.milkomeda.com</p>
> 
> Chain ID: 200101
> 
> Currency Symbol (Optional): milkTADA
> 
> <p> Block Explorer URL (Optional): https://explorer-devnet-cardano-evm.c1.milkomeda.com </p>
WebSocket 엔드포인트가 필요한 경우 다음을 사용하십시오. wss://rpc-devnet-cardano-evm.c1.milkomeda.com

:::info

Milkomeda C1 메인넷 버전을 사용하는 경우 매개 변수는 다음을 입력합니다.

- Network Name: Milkomeda Cardano (C1)
- <span>New RPC URL: https://rpc-mainnet-cardano-evm.c1.milkomeda.com</span>
- Chain ID: 2001
- Currency Symbol: milkADA
- <span> Block Explorer URL: https://explorer-mainnet-cardano-evm.c1.milkomeda.com </span>

WebSocket 엔드포인트가 필요한 경우 다음을 사용하십시오. wss://rpc-mainnet-cardano-evm.c1.milkomeda.com:::

「**Save**(저장)」를 클릭하면 네트워크 드롭다운에 「**Milkomeda Cardano Testnet**」이 표시됩니다. 아래와 같이 계정 이름을 클릭하여 Milkomeda Cardano Testnet 지갑 주소를 클립보드에 복사합니다.

![Metamask 복사 주소](/img/metamask-copy-c1-testnet-address.png)

다음으로 Flint 지갑으로 돌아가서 테스트 ADA를 래핑합니다.
