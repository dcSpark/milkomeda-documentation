# 플린트 지갑 구성

Flint Wallet은 DeFi 및 NFT를 위한 라이트지갑 역할을 하는 Chrome 확장 프로그램입니다. 현재 버전의 Flint를 사용하면 사용자가 Cardano Testnet 지갑을 만들고 "Milkomeda 모드"를 활성화하여 Milkomeda C1 사이드체인으로 트랜잭션을 보낼 수 있습니다.

Chrome 브라우저를 사용하는 동안 다음 링크를 방문하여 "Chrome에 추가" 버튼을 클릭하여 Flint Wallet을 설치합니다.

플린트 지갑 설치< /a></p> 

Chrome 브라우저를 사용하는 동안 다음 링크를 방문하여 "Chrome에 추가" 버튼을 클릭하여 Flint Wallet을 설치합니다. 플린트 지갑 설치< /a>설치 후 Chrome 창 오른쪽 상단에 있는 확장 프로그램 아이콘 <img src={puzzleIcon} className="img-icon" />을 클릭하고 **플린트 지갑**. Flint 창 내에서 초기 경고 화면을 지나 "**시작하기**"를 클릭하고 언어를 선택하고 "**동의합니다 **"를 입력하고 아래와 같은 화면이 나타나면 "**Testnet**" 버튼을 클릭합니다.</p> 

:::정보
Milkomeda C1의 메인넷 버전에서 실제 ADA를 사용하려면 "메인넷"을 클릭하여 메인넷 Cardano 지갑을 만드십시오.
:::

![테스트넷 네트워크선택](/img/flint-select-testnet.png)

그런 다음 "**확인**", "**새 지갑 만들기**"를 클릭하고 복구 구절 사본을 기록해 둡니다.

다음 화면에서 복구 구절을 확인하고 지갑 비밀번호를 설정한 후 “**내 지갑 만들기**”를 클릭합니다. 이제 앱 암호 화면에서 "**건너뛰기**"를 선택한 다음 "**완료**"을 클릭할 수 있습니다. 다음과 같이 Cardano Testnet 지갑이 생성됩니다.

![Flint 홈 화면](/img/flint-home-screen.png)

다음으로 Cardano Testnet 포셋에 접속하여 테스트 ADA를 취득합니다.
