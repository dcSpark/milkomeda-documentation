# Test ADA를 취득

Cardano Testnet 포셋에서 테스트 ADA를 받을 준비가 되었습니다.

:::info

Milkomeda C1 메인넷 버전에서 실제 ADA를 사용하는 경우 이 단계를 건너뛰고 [다음 페이지](./configuring-metamask-for-devs)로 이동합니다.

:::

Flint 월렛 창에서 **Receive**(받기) 를 클릭합니다. 「Receive(받기) 」창에서 **Copy Address**(주소 복사) 를 클릭하여 Cardano Testnet 지갑 주소를 클립보드에 복사한 다음, 화면 하단에서 「**Testnet faucet(tADA)**(테스트넷 포셋(tADA))」을 클릭합니다.

![](/img/flint-copy-tada-receive-address.png)​

그러면 다음 링크로 이동합니다. [Cardano Testnet 포셋](https://testnets.cardano.org/en/testnets/cardano/tools/faucet/)

이 화면의 드롭다운 메뉴에서 「tAda」를 선택하고 「Address(주소)」 필드에 Cardano Testnet 월렛 주소를 붙여넣습니다. 「API Key」란은 비워 둘 수 있습니다.

![](/img/cardano-testnet-faucet.png)

Captcha를 완료한 후 「**Request funds**(자금 요청)」을 클릭합니다.

몇 분 정도 기다리면 (보통 약5분) Cardano Testnet 지갑에 tADA가 표시됩니다.

![](/img/flint-confirm-tada-balance.png)

그런 다음 MetaMask를 설정하여 Milkomeda C1 사이드 체인의 주소를 가져옵니다.
