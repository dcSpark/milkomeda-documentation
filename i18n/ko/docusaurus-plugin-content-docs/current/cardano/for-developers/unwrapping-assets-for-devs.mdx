# 자산 래핑 풀기

Milkomeda Cardano Unwrap Bridge는 자산을 Milkomeda Cardano Testnet에서 Cardano 메인넷으로 이동하기 위한 도구입니다. 아래 단계에 따라 MilkTADA의 래핑을 풀고 카르다노 테스트넷 지갑에서 tADA를 받으세요.

먼저 [Devnet용 Milkomeda Cardano Unwrap Bridge](https://devnet-cardano-bridge.milkomeda.com/bridge)에 액세스하여 MetaMask 지갑을 연결합니다.

![](/img/bridge-connect-metamask.png)

![](/img/bridge-connect-metamask-tada-confirm-1.png)

![](/img/bridge-connect-metamask-tada-confirm-2.png)

다음으로, 래핑을 풀기 위해 "To" 필드에 Cardano Testnet 지갑 주소를 붙여넣고 milkADA의 수량 입력합니다. 그런 다음 "다음 단계" 버튼을 클릭합니다.

![](/img/bridge-enter-tada-amount.png)

MetaMask에서 거래를 확인합니다.

![](/img/bridge-unwrap-tada-confirm.png)

필요한 거래 확인을 위해 약 3~5분 정도 기다린 후 Cardano Testnet 지갑에 포장되지 않은 tADA가 표시되어야 합니다.
