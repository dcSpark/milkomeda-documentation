# 플린트 지갑 구성

Flint 지갑은 DeFi 및 NFT를 위한 라이트지갑 역할을 하는 Chrome 확장 프로그램입니다. Flint의 현재 버전에서는 "Milkomeda 모드"를 활성화하여 Milkomeda C1 사이드체인에 거래를 보낼 수 있습니다. Chrome 브라우저를 사용하는 동안 다음 링크를 방문하여 "Chrome에 추가" 버튼을 클릭하여 Flint 지갑을 설치합니다.

설치가 완료되면 Chrome 창 오른쪽 상단의 '퍼즐 조각' 아이콘 <img src={puzzleIcon} className="img-icon" />을 클릭하여 확장 프로그램에 액세스합니다.. **Flint**를 선택합니다. Flint 창 내에서 초기 경고 화면 다음으로 표시되는 "**시작하기**"를 클릭 후 언어를 선택하고 "**동의합니다 **"를 입력한 후 아래와 같은 화면이 나타나면 "**Mainnet**" 버튼을 클릭합니다.

![네트워크선택](/img/pick-a-network.png)

그 다음 "**확인**", "**새 지갑 만들기**"를 클릭하고 복구 구절 사본을 기록해 둡니다.

다음 화면에서 복구 구절을 확인하고 지갑 비밀번호를 설정한 후 “**내 지갑 만들기**”를 클릭합니다. 지금은 앱 암호화면에서 "**건너뛰기**"를 선택한 후 "**완료**"을 클릭해도 문제없습니다. 다음과 같이 Cardano 지갑이 생성됩니다.

![Flint지갑 생성](/img/create-cardano-wallet.png)

다음 단계로 진행하려면 지갑에 ADA가 있어야 합니다. 도움이 필요하거나 이 단계에서 문제가 발생하는 경우 이 문서의 마지막 '[지원](../../home/get-help)' 섹션을 참조하세요.

그런 다음 MetaMask를 설정하여 Milkomeda C1 사이드체인에 연결합니다.
