# Cardano Bridge가 지원하는 네이티브 애셋

이 섹션에서는 Cardano에 대한 Milkomeda Bridge가 지원하는 네이티브 토큰을 모두 소개합니다. 지원되는 네이티브 애셋을 랩핑하거나 랩핑 해제하려면 [최종 사용자용](./) 섹션의 _단계별</ em> 자습서를 기반으로 `milkADA`를 아래 표에 표시된 토큰으로 바꿉니다.</p>

:::info
특정 네이티브 에셋은 Milkomeda Token Registry에 추가한 다음 랩핑 및 랩핑 해제 작업을 수행해야 합니다. 따라서 다음 목록에는 제한이 있으며 특정 네이티브 토큰에 대해서만 허용됩니다.
:::
