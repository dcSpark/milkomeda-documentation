# 메타마스크 설치

다음 단계는 MilkADA를 받을 수있는 Milkomeda C1 사이드체인의 주소를 취득합니다. 현 단계에서는 MetaMask를 사용하는 것이 좋습니다.

먼저 [Chrome용 MetaMask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn)를 설치합니다. 그 다음에 Chrome 확장 프로그램 아이콘 <img src={puzzleIcon} className="img-icon" />을 클릭하고 MetaMask를 선택합니다. MetaMask 창에서 **Get Started**(시작) 를 클릭한 후 **Create a Wallet</>(지갑 만들기) 를 선택합니다. 지갑 비밀번호를 설정합니다. 이 후에 설정 화면이 표시되므로 클릭하여 계속 진행합니다. 완료되면 아래와 같은 빈 지갑이 새로 생성됩니다. MetaMask 화면 오른쪽 상단의 네트워크 드롭다운을 선택합니다.

![MetaMask지갑](/img/metamask-wallet.png)

드롭다운 목록에서 **Add Network**(네트워크 추가)를 선택하고 다음 내용을 입력합니다.

> Network Name: Milkomeda Cardano (C1)
> 
> <p> New RPC URL: https://rpc-mainnet-cardano-evm.c1.milkomeda.com </p> 
> 
> Chain ID: 2001
> 
> Currency Symbol (Optional): MilkADA
> 
> <p>Block Explorer URL (Optional): https://explorer-mainnet-cardano-evm.c1.milkomeda.com </p>
「**Save**(저장)」를 클릭하면 네트워크 드롭다운에 「**Milkomeda Cardano (C1)**」이 표시됩니다. 아래와 같이 계정 이름을 클릭하여 C1 지갑 주소를 클립보드에 복사합니다.

![MetaMask주소 복사](/img/copy-address-metamask.png)

이제 Flint 지갑으로 돌아가서 ADA를 보냄으로써 MilkADA로 변환할 수 있습니다. &#x20;
