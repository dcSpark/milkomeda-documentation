# MilkADA 취득

Flint 지갑 창을 열고 **Send**(보내기) 를 클릭합니다. 다음 화면에서 '**Milkomeda 모드**"의 토글을 전환하여 활성화합니다. 클립보드에 복사한 Milkomeda C1 지갑 주소를 '**Milkomeda Ethereum Address**' 입력란에 붙여넣고 송금할 금액을 입력합니다(예에서는 1Test ADA 송금하고 있습니다). 「**Continue**（계속）」을 클릭합니다.

![플린트 지갑에서 ADA를 래핑](/img/wrapping-send-flint.png)

확인 화면에서 지갑 비밀번호를 입력하고 **Send**(보내기) 를 클릭합니다.

![ADA를 래핑합니다](/img/wrapping-ada-send.png)

송금에는 확인을 10회 수행해야 하며 약 **5**분이 소요됩니다. 거래를 확인하면 제출한 MilkADA가 MetaMask의 C1 지갑에 표시됩니다.

:::info 거래 수수료에 대해 Milkomeda C1 사이드체인에서만 수행되는 거래의 경우 블록생성자가 설정한 표준 Milkomeda 네트워크 거래수수료만 지불합니다.

다만 래핑이 포함된 거래(Cardano와 Milkomeda C1 사이드체인 간에 자산이 이동하는 거래)의 경우 하나의 거래가 각 체인에서 이루어지므로 Cardano와 Milkomeda 거래 수수료를 모두 지불해야 합니다.

예를 들어 Cardano에서 Milkomeda C1로 자산을 보내는 경우 Cardano에서 자산을 잠그는 거래 하나와 Milkomeda C1에서 래핑된 자산을 발행하기 위한 거래 하나가 필요합니다.

Flint 지갑에서 이와 같은 트랜잭션을 보내는 경우 "Network Fee(네트워크 수수료)" 항목을 확장하여 "Base Fee(기본 수수료)"(Cardano 트랜잭션 수수료) 및 "Milkomeda Fee(Milkomeda 수수료)" 를 확인할 수 있습니다. 이러한 거래의 'Milkomeda Fee(Milkomeda 수수료)'는 자산을 래핑하는 역할을 담당하는 블록생성자에게도 보상을 지불하게 되기 때문에 Milkomeda에서의 표준 거래 수수료보다 약간 높아지게 되는 것에 주의하시길 바랍니다. :::

이상으로 Milkomeda C1 사이드체인에서 사용하기 위해 ADA를 MilkADA로 래핑할 수 있었습니다.

:::주의 "MilkADA"와 "Wrapped ADA" MilkADA는 Milkomeda C1 사이드체인에 기반한 자산입니다. ADA를 Milkomeda로 이동시키면 MilkADA를 받을 수 있습니다.

MilkADA 외에도 "Wrapped ADA" 또는 "WADA"라는 토큰이 있습니다. Wrapped ADA는 Ethereum의 WETH와 유사한 기능을 수행하는 ERC20 토큰입니다. 이들을 혼동하지 않도록주의하십시오. :::

:::info 래핑할 수 있는 토큰 목록은 "[**Cardano Bridge에서 지원하는 네이티브 애셋**](./cardano-bridge-supported-native-assets)' 섹션을 확인하십시오. :::
