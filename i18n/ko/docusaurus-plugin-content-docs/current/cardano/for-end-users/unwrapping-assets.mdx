# 자산 래핑 풀기

자산을 Milkomeda C1 사이드체인에서 Cardano로 되돌리려면 Milkomeda Cardano Unwrap Bridge를 사용합니다. 아래 단계에 따라 milkADA의 래핑을 풀고 Cardano 메인넷 지갑에서 ADA를 받으세요. &#x20;

우선 [Milkomeda Cardano Unwrap Bridge](https://cardano-bridge.milkomeda.com/bridge)로 이동하여 MetaMask 지갑을 연결합니다.

![MetaMask 접속하기](/img/connect-metamask.png)

![지갑 연결하기](/img/connect-wallet.png)

Next(다음) 를 _클릭_합니다.

![Milkomeda네트워크 접속하기](/img/connect-milkomeda.png)

_Milkomeda C1 사이드체인에 연결_

:::주의 지갑의 적절한 네트워크를 선택합니다. <br /> 신뢰할 수 없는 사이트에 연결하면 안 됩니다. 웹사이트 URL이 다음과 같은지 확인하십시오: https://cardano-bridge.milkomeda.com :::

그런 다음 래핑을 해제할 milkADA의 금액을 입력하거나 화살표를 클릭하여 래핑을 해제할 다른 자산을 선택하고 'To' 입력란에 자신의 Cardano 지갑 주소를 붙여넣습니다. 그 다음 "다음 단계" 버튼을 클릭합니다.

<img className="img-full" src={tokenSelection} />

「Confirm(확인)」을 클릭하여 MetaMask에서 자산을 래핑 해제할 트랜잭션에 서명합니다.

![MetaMask를 확인하기](/img/confirm-metamask.png)

블록체인 확인을 기다립니다.

<img className="img-full" src={blockchainConfirmation} alt="블록체인 확인하기" />

:::info

거래에 대한 자세한 내용은 [Cardanoscan](https://cardanoscan.io) 탐색기에서 확인할 수 있습니다.

:::

이 시점에서 브라우저의 주소 표시줄에는 래핑 해제 거래에 대한 정보가 포함되어 있습니다. 다음 그림은 예를 보여줍니다.

<img className="img-full" src={addressBar} alt="주소 표시줄 링크" />

주소 표시줄의 요소의 의미는 각각 다음과 같습니다.

- EVM의 랩핑 해제 트랜잭션 ID
- 래핑 해제된 토큰 금액
- 래핑이 해제된 토큰 이름

래핑 해제 거래에 대한 자세한 내용은 [Milkomeda Blockscout 탐색기](https://explorer-mainnet-cardano-evm.c1.milkomeda.com/)를 방문하여 다음과 같이 검색창에 EVM 랩핑 해제 거래 ID를 붙여넣습니다.
