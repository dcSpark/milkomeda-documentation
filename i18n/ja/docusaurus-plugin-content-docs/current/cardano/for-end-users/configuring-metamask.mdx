# MetaMaskの設定

The next step is to obtain an address on the Milkomeda C1 sidechain where we can receive our MilkADA. 現時点では、このステップにはMetaMaskを使用することをおすすめしています。

まず、[Chrome用MetaMask](https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn)をインストールします。 Then, click the Chrome Extensions icon <img src={puzzleIcon} className="img-icon" /> and select MetaMask. MetaMaskのウィンドウ内の「**Get Started**（開始する）」をクリックして、次に「**Create a Wallet**（ウォレットを作成）」を選択します。 ウォレットのパスワードを設定します。この後に設定画面が表示されるので、クリックして先に進みます。 When finished, you should have a new empty wallet as shown below. MetaMask画面の右上のネットワークドロップダウンを選択します。

![MetaMask wallet](/img/metamask-wallet.png)

ドロップダウンリストから「**Add Network**（ネットワークを追加）」を選択して、以下の内容を入力します。

> Network Name: Milkomeda Cardano (C1)
> 
> <p> New RPC URL: https://rpc-mainnet-cardano-evm.c1.milkomeda.com </p>
> 
> Chain ID: 2001
> 
> Currency Symbol (Optional): MilkADA
> 
> <p>Block Explorer URL (Optional): https://explorer-mainnet-cardano-evm.c1.milkomeda.com </p>
Click “**Save**”, and you should now see "**Milkomeda Cardano (C1)**" in the network dropdown. Copy your C1 wallet address to the clipboard by clicking on the account name, as shown.

![Copy Address in MetaMask](/img/copy-address-metamask.png)

Now we are ready to return to Flint Wallet and send our ADA to be converted to MilkADA. &#x20;
