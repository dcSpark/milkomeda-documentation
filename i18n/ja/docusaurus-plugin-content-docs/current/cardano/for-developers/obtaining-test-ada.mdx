# Test ADAの取得

Cardano TestnetフォーセットからテストADAを受け取る準備ができました。

:::info

Milkomeda C1メインネットバージョンでリアルADAを使用する場合は、この手順を飛ばして[次のページ](./configuring-metamask-for-devs)に進みます。

:::

Flintウォレットのウィンドウで「**Receive**（受け取る）」をクリックします。 「Receive（受け取る）」ウィンドウで「**Copy Address**（アドレスをコピー）」をクリックしてCardano Testnetウォレットのアドレスをクリップボードにコピーし、以下の通り、画面下部の「**Testnet faucet (tADA)**（テストネットフォーセット（tADA））」をクリックします。

![](/img/flint-copy-tada-receive-address.png)​

これにより、次のリンク先に移動します。 [Cardano Testnetフォーセット](https://testnets.cardano.org/en/testnets/cardano/tools/faucet/)

この画面でドロップダウンメニューから「tAda」を選択し、「Address（アドレス）」欄にCardano Testnetウォレットのアドレスを貼り付けます。 「API Key」欄は空白のままで構いません。

![](/img/cardano-testnet-faucet.png)

Captchaを完了したら、「**Request funds**（資金をリクエストする）」をクリックします。

数分待つと（通常は5分程度）、Cardano TestnetウォレットにtADAが表示されます。

![](/img/flint-confirm-tada-balance.png)

次にMetaMaskを設定して、Milkomeda C1サイドチェーンのアドレスを取得します。
