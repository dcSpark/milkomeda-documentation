# 資産のラッピング

**Algorand（メインチェーン）からMilkomeda A1ロールアップにアセットを転送**するためには、専用のブリッジdAppを使用する必要があります。 A1ロールアップにアセットをラッピングするには、Milkomeda A1ブリッジdApp（[https://algorand-bridge.milkomeda.com/](https://algorand-bridge.milkomeda.com/)）にアクセスしてください。

アセット・ラッピングを開始するには、「Network Origin（送信元ネットワーク）」ドロップダウン・メニューから**AlgorandからMilkomeda**への方向を選択します。 次に、「Select Token（トークン選択）」ドロップダウン・メニューから、ラッピングしたいトークン（このチュートリアルでは、ALGOを使用します）を選択します。

<img src={dappselectorigin} className="img-full" alt="ラッピングの確認" />

次に、MyAlgoウォレットとMetaMaskウォレットにログインします。

1. 送信元側の「Connect Wallet（ウォレット接続）」をクリックすると、MyAlgoウォレット（最近Peraウォレットもサポートされました）が、パスワードの入力とdAppに接続するアカウントの確認を要求します。 利用可能なALGOの残高は、「Select Token（トークン選択）」欄の下に表示されます。

2. dAppの送信先側にある「Connect Wallet（ウォレット接続）」をクリックすると、Metamaskウォレットはネットワークを（まだ接続していない場合は）Milkomeda A1ネットワークに切り替えるかどうか確認を求めてきます。

A1ロールアップに転送したい**ALGOの数を入力**します（残高全体を送信したい場合は「Max（最大）」をクリックすると、ブリッジが送信できる最大額から取引手数料を差し引いた額を計算します）。

**「Next（次へ）」をクリック**すると、「Estimated Fees（予想手数料）」と「Estimated Wrapping Confirmation Time（予想ラッピング確認時間）」を含む取引概要が表示されます。

<img src={wrappingsummary} className="img-full" alt="ラッピング・サマリー" />

<strong x=id="1">「Sign and Send（署名して送信）」をクリック</strong>し、ウォレット・パスワードを入力し、「Confirm（確認）」をクリックしてください。 あなたのトークンがブリッジに送られ、ラッピングされ、Milkomeda Algorand A1ロールアップのあなたのMetaMaskウォレットに送られるのを見守りましょう。 パスワードの確認に約20秒かかる場合があります。

:::情報
ロールアップのトランザクションはAlgorandブロックチェーンに書き込まれることを忘れないでください。 つまり、1つの確認はウォレットからブリッジ・アドレスに資金を送信するためのもので、2つ目の確認はAlgorand上のロールアップ・バッチ・トランザクションのためのものなのです。
:::

送信に必要な確認は全部で2回で、約20秒かかります（Algorandでは4.5秒、Rollupのバッチでは約14秒かかります）。 トランザクションが確認されると、送金されたMilkALGOはMetaMaskのMilkomeda Algorandウォレットに表示されるはずです。

<img src={wrappingcompleted} className="img-full" alt="ラッピング完了" />

これで、トランザクションがブリッジを通過する様子を見ることができます。

<img src={wrapsuccess} className="img-full" alt="ラッピング成功ポップアップ" />

**おめでとうございます！**あなたは今、Ethereumエコシステム以外の最初のロールアップにあなたのトークンをラッピングしました！
