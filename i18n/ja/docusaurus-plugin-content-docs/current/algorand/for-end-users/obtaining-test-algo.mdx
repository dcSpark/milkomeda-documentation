# Algorandウォレットの設定とテストALGOの取得

**Milkomeda Algorandロールアップ・ブリッジ**と相互作用するためには、ネイティブAlgorandウォレットを設定する必要があります。 Milkomeda A1ブリッジと互換性のあるテストネット・ウォレットを設定するために、MyAlgoウォレットを使用することをお勧めします。

### MyAlgoウォレットの設定

まず、[https://wallet.myalgo.com/new-account](https://wallet.myalgo.com/new-account) に行きます。 _利用規約_に同意した後、「_Continue（続ける）_」をクリックし、ウォレットの設定を開始します。 次のステップでは、パスワードを作成するよう求められます。 次に、**ランダムに生成される新しいアカウントを作成**してください。

<img src={myAlgoWallet} className="img-full" alt="" />

次のステップでは、ニーモニックフレーズを保存するかどうかを尋ねられます。

<img src={mnemonicphrase} className="img-full" alt="" />

:::情報
以下のニーモニックフレーズの単語を正しい順番で覚えておいてください。 これらは、あなたの新しいアカウントのリカバリーフレーズとなります。
:::  
:::警告
これらの単語を誰とも共有しないでください。 これらの単語は、あなたの資金を盗むために使われる可能性があります。
:::  
ニーモニックフレーズを確認したら、あなたのMyAlgoウォレットは設定され、あとはお好きな名前をつけるだけです。

### テストALGOの取得

ブリッジdAppと対話するためには、テストネットのALGOトークン、ALGOを取得する必要があります。 はじめに、公式のテストネット・ディスペンサー [https://testnet.algoexplorer.io/dispenser](https://testnet.algoexplorer.io/dispenser) からTestnet ALGOを取得することをおすすめします。

<img src={algoDispenser} className="img-full" alt="" />

新しく作成したAlgorandアドレスを入力し、数分（通常2分以内）待つと、Algorandテストネット・ウォレットにいくつかのtALGOが表示されるようになるはずです。

<img src={algoWalletDashboard} className="img-full" alt="" />

次に**MetaMask Wallet**を設定し、**Milkomeda A1 Algorandロールアップ**でアドレスを取得することにします。
