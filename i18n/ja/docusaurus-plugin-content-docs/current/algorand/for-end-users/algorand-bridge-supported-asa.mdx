# A1ロールアップ・ブリッジでサポートされたASA

このセクションでは、AlgorandへのMilkomedaブリッジでサポートされているすべてのAlgorand標準アセットを紹介します。 サポートされているネイティブ・アセットをラッピング、アンラッピングするには、[エンドユーザ向け](./)セクションで紹介されている_ステップバイステップ_のチュートリアルに沿って、`milkAlgo`を以下のテーブルから任意のトークンに置き換えてください。

:::情報
ラッピングやアンラッピングを行う前に、特定のネイティブ・アセットがMilkomedaトークン・レジストリに追加されている必要があります。 したがって、以下のリストには制約があり、特定のネイティブトークンについてのみ許容されます。
:::
:::

:::注意
Milkomedブリッジでラッピング／アンラッピングできるトークンの最新リストを見たい場合は、[Bridge Explorer](https://bridge-explorer.milkomeda.com/algorand-mainnet/tokens)ですべてのトークンを見ることができます（あなたのMetamaskに追加するためのボタンもあります）。
