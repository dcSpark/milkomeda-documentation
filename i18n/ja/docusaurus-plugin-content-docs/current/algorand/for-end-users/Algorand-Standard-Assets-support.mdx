# Algorand標準アセット（ASA）の移動

ここでは、Milkomeda A1ロールアップ・ブリッジがサポートしているAlgorand標準アセット（ASA）について簡単に紹介し、Milkomeda A1ロールアップとメインチェーン（Algorand）間でASAを移動（ラッピングまたはアンラッピング）する方法について説明します。 Milkomeda A1ブリッジを使ったASAのラッピングとアンラッピングの方法を順を追って説明する前に、ASAとは何かを理解することが重要です。

### Algorand標準アセット(ASA)

Algorand標準アセット（ASAs）は、Algorandがサポートするオンチェーン資産で、ネイティブ通貨であるAlgoと同じセキュリティ、互換性、速度を保証するものです。 ASAは、ステーブルコイン、ゲーム内通貨、ロイヤリティ・ポイントなどを表すことができます。 また、収集品のような単一のアセットを表すこともできます。

:::情報
同じ種類の複数の資産であるASAは「Fファンジブル・アセット」と呼ばれ、例えばステーブルコインがそうです。 ユニークであるものは、ノン・ファンジブル・アセットと呼ばれます。 <br /> Learn more about ASAs: https://developer.algorand.org/docs/get-details/asa/
:::

それでは、Algorand標準アセットをAlgorandウォレットに追加し、Milkomeda A1ブリッジを使って移動させる方法を説明します。

### AlgorandウォレットにASAを追加する方法

:::情報
ASAを受け取るためには、まずアカウントにそのアセットをオプトインする必要があります。 どのアカウントでもAlgorand標準アセットを受け取ることができますが、特定のASAを受け取る可能性がある場合は、まず**その資産にオプトイン**する必要があります。 アセットへの「オプトイン」は、オプトインしたアカウントへの、またはアカウントからの0件のアセット転送にすぎません。 このプロセスでは、資産を譲渡する前に、特定の資産をアカウントに追加することになります。
:::

##### EUReをMyAlgoウォレットにオプトインするためのステップバイステップ・ガイド

1. MyAlgoウォレットを開きます。

2. 下の画像で強調されているように、右側の「Add Asset（アセット追加）」ボタンを押します。

<img src={addasset} className="img-full" alt="アセット追加" />

3. ASA IDまたはそのシンボル（12400859またはEURE）を入力します。

<img src={addingEuretowallet} alt="ウォレットにEURe追加" />

> この特定のASAを見つけるのに苦労している場合、あなたのウォレットがAlgorandのテストネット（_右上隅_）に接続されていることを確認してください。

4. アカウント内のアセット・リストに値0のEUReが表示されるまで待ちます。

これで、あなたのアカウントはテストネット・ディスペンサーからEUReを受け取ることができます。

### テストネットEUReを取得する方法

このチュートリアルの目的では、Algorand公式のテストネット・ディスペンサーから利用できるデフォルトのASAであるテストネットEurosにのみ焦点を当てます。

以下の手順で、最初のテストネットEUResを取得してください。

1. [https://testnet.algoexplorer.io/dispenser ](https://testnet.algoexplorer.io/dispenser)にアクセスします。
2. Select "Testnet Euros" from the drop-down list, as shown in the image below. <img src={asadispenser} className="img-full" alt="ディスペンサーからASA取得" />
3. Algorandのウォレット・アドレスを貼り付けます。
4. 必要なアンチボット認証を完了します。
5. 「Dispense（取り出す）」ボタンをクリックし、テストネットEurosを受け取ります。

テストネットEurosを受け取ったら、Milkomeda | Algorand A1ロールアップ・ブリッジとのやりとりを開始する準備が整いました。 クール！

### MilkomedaブリッジでのAlgorand標準アセットのラッピング

:::注意
このチュートリアルでは、プロセスを最もよく見せるために、`EURe`というテストネットASAを使用します。 メインネット・アセットをラッピングもしくはアンラッピングしたい場合は、EUReをMilkomeda A1メインネット・ブリッジでサポートされている他のトークンに置き換えて、このチュートリアルを使用してください。 利用可能なアセットの一覧は「[AlgorandブリッジがサポートするASA](algorand-bridge-supported-asa.mdx)」を参照してください。

Milkomeda A1ブリッジdApp（[https://algorand-bridge-dev.milkomeda.com/](https://algorand-bridge-dev.milkomeda.com/)）にアクセスします。 アセットのラッピングを開始するには、「Network Origin（送信元ネットワーク）」ドロップダウン・メニューから「**Algorand to Milkomeda direction（AlgorandからMilkomedaへ）**」を選択します。 次に、「Select Token（トークン選択）」ドロップダウン・メニューから**適切なトークン (EURe) **を選択します。

<img src={EURewrappingchoice} className="img-full" alt="EUReをラッピング" />

:::警告
EUReはメインチェーンでの取引手数料の支払いに使われるため、EUReを動かすにはAlgorandウォレットに最低限のALGOトークンが入っている必要があることを覚えておいてください。
:::

A1ロールアップに転送したい**テストEUReの数を入力**します。

「**Next（次へ）**」をクリックすると、「Estimated Fees（予想手数料）」と「Estimated Wrapping Confirmation Time（予想ラッピング確認時間）」が記載された取引概要が表示されます。

<img src={EURewrappingconfirmation} className="img-full" alt="EURラッピングを確認" />

**「Sign and Send（署名して送信）」をクリック**し、MyAlgoウォレットのパスワードを入力し、トランザクションを確認したら、あなたのトークンがブリッジに送られ、ラッピングされ、Milkomeda Algorand A1ロールアップ上のあなたのMetaMaskウォレットに送られます。

:::情報
転送には合計2回の確認が必要で、約20秒かかります（Algorandが4.5秒、ロールアップのバッチが約14秒）。
:::

これで、あなたのトランザクションがブリッジを通過するのを見ることができます。

<img src={ASAwrappedconfirmations} className="img-full" alt="ASAラッピング・ブロックチェーン確認" />

トランザクションが確認されたら、ASAをMetaMaskウォレットに追加してください。

ネットワーク確認セクションの下にある**「Import Assets to Metamask（Metamaskにアセットをインポート）」ボタンを使って**、あなたのMetamaskウォレットにラッピングされたEUReを確認します。 このボタンでMetaMaskウォレットが表示され、下の画像のように、ラッピングされたASAをウォレットに追加するよう提案されます。

<img src={AddASSAtoMetamask} alt="MetamaskにASA追加" />

**「Add Token（トークン追加）」**をクリックすると、MetaMaskウォレットにラッピングされたASAを表示できるようになります。

### MilkomedaブリッジでAlgorand標準アセットをアンラッピングする

Milkomeda A1ブリッジdApp（[https://algorand-bridge-dev.milkomeda.com/](https://algorand-bridge-dev.milkomeda.com/)）にアクセスします。 アセットをアンラッピングするために、「Network Origin（送信元ネットワーク）」ドロップダウン・メニューから「**Milkomeda to Algorand ddirection（MilkomedaからAlgorandへ）**」を選択します。 次に、「Select Token（トークン選択）」ドロップダウン・メニューから**適切なトークン (EURe) **を選択します。

<img src={ChooseASAtounwrap} className="img-full" alt="アンラッピングするASA選択" />

Algorandに戻すテストEUReの数を入力します。 「**Next（次へ）**」をクリックすると、「Estimated Fees（予想手数料）」と「Estimated Wrapping Confirmation Time（予想ラッピング確認時間）」が記載された取引概要が表示されます。

:::注意
EUReを移動するには、MetaMaskウォレットに最低1つのテスト用milkALGOがある必要があります。milkALGOはアンラッピング時にブリッジ手数料が支払われる通貨だからです。
:::

「**Sign and Send（署名して送信）**」をクリックすると、あなたのMetaMaskウォレットが表示されます。 あとは、ERC-20の許可を承認する必要があります。

:::情報
ERC20許可を与えるとは、簡単に言うと、ユーザーが他のコントラクト/アドレスにERC20トークンを転送する許可を与えるということです。 つまり、他のアドレスがトークンを使うことを許可するのです。
:::

<img src={erc20allowance} alt="ASAアンラッピングの確認" />

MetaMaskでERC-20許可が確認された後、dAppはEVMから許可の承認を受けるのを待ちます。

<img src={tokenerc20allowance} className="img-full" alt="ASAアンラッピングの確認" />

このシナリオでは、ブリッジ・コントラクトによるERC20トークン（例：EURe）の転送を承認しています。 その結果、コントラクトはEVM上のユーザーのアカウントからブリッジにトークンを移動することができ、そこでアンラッピングが実行されます。

**ERC20の許可が承認されると、アンラッピング要求の提出に進むことができます。** MetaMaskがもう一度ポップアップして、要求されたトランザクションの承認を求めます。

<img src={unwrappingreqconfirmation} alt="ASAアンラッピングの確認" />

ASAアンラッピング手順の最後のステップでは、アンラッピング要求を処理するために、1 MILKALGOアンラッピング手数料をバリデーターに支払う必要があります。

**おめでとうございます！**これでMilkomeda AlgorandロールアップからAlgorandへASAのラッピングを解除することができました！
